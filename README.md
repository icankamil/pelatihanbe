## Tentang Project

Project ini backend dari react app di repo pelatihan fe. Endpointnya :
Insert data : ```POST | http://{host}/api/pelatihan```
Get All Data : ```GET | http://{host}/api/pelatihan```
Get Single Data : ```GET | http://{host}/api/pelatihan/{id}```
